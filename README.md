# Todo App

Uma aplicação de gestão de tarefas desenvolvida com HTML, CSS e JavaScript.

## Funcionalidades

- Adicionar tarefas
- Editar tarefas
- Remover tarefas
- Marcar tarefas como concluídas
- Filtros:
  - Todas
  - Pendentes
  - Concluídas

- Guardar tarefas com localStorage

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- LocalStorage

## Objetivo do Projeto

Este projeto foi criado para praticar:

- Manipulação do DOM
- Eventos
- Estado da aplicação
- Organização de código em módulos
- Renderização dinâmica
- CRUD operations
